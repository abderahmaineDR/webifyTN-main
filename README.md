# WebifyTN

WebifyTN is a modern, responsive agency/portfolio website built with React, TypeScript, Vite, and Tailwind CSS.
It showcases services, projects, and contact information with multilingual support and smooth UI animations.

## Features

- Multi-page routing (`Home`, `About`, `Services`, `Portfolio`, `Contact`)
- Multilingual interface (`French`, `English`, `Arabic`)
- Dark/light theme toggle
- Responsive layout for desktop and mobile
- Reusable UI components based on Radix + shadcn patterns
- Smooth motion effects using Framer Motion

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Framer Motion

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Start development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```

## Project Structure

```text
src/
  components/    # Shared UI and section components
  contexts/      # Theme and language providers
  hooks/         # Custom hooks
  locales/       # Translation content
  pages/         # Route pages
  assets/        # Images and static media
```
